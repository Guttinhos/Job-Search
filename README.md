# Job-Search
Clone da pagina de empregos do google utilizando vue

## Utilizando: 
- Vue
- Pinia
- Vue Router
- VueX
- Vitest
- Tailwind CSS
- TypeScript

## Instalação

```sh
npm install
```

### Compilar e rodar a aplicação

```sh
npm run dev
```

### Compilação para produção

```sh
npm run build
```

### Rodar testes unitários com o [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```

### Lint Com [ESLint](https://eslint.org/)

```sh
npm run lint
```
